# React Dashboard

A personal dashboard built with React.

## Features
- Time Widget
- Weather widget
- Todo list
- Random Quote Widget
- Theme Changer
- LocalStorage persistence

## Tech Stack
- React
- JavaScript
- CSS
- OpenWeather API
- Quoteshub API

## Setup
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file based on `.env.example`
4. Run `npm run dev`
